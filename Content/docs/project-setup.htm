Project Setup

  1. Create a new MVC 3 Razor project called TodoApp using the Really Empty project template:
    • Project Template Article
    • Visual Studio Installer
  2. Add a reference to EntityFramework via NuGet: Entity Framework 4.3.1
  3. Add a reference to DotNetOpenAuth via NuGet: DotNetOpenAuth (unified)
  4. Add a reference to JQuery via NuGet: jQuery
  5. Add a reference to JQuery UI via NuGet: jQuery UI
  6. Add a reference to Exo via NuGet: Exo for EF4 and MVC3
